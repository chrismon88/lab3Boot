Finding the area of a rectangle.

One of the first programs I learned in python was an area calculator. Below is a sample of code for the program.


  #rectangle area
#rectangle input
length= float(input("What is the length of the rectangle's side?"))
width=float(input("What is the width of the rectangle's side?"))
# area calculation
area=length*width
#area output formatted
print("The rectangles area is",
    format(area,',.2f'))
